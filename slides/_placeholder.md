# Thư mục slides/

Đặt file slide trình bày vào đây khi hoàn thành, ví dụ:

- `Slide_trinh_bay_DeTai43_231A010880.pptx` (hoặc `.pdf`)

Yêu cầu: 8-12 trang, theo mẫu slide ở mục 8 tài liệu hướng dẫn (tên đề
tài/nhóm, vấn đề & mục tiêu, công cụ GitHub dùng, mô hình/quy trình,
kết quả chính, phân tích rủi ro, đánh giá/hạn chế, kết luận, tài liệu
tham khảo + link repo).

xóa sau khi đã thêm slide thật vào đây.)*
